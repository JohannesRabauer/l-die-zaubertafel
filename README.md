# Die Zaubertafel 🖍️

Ein interaktives Mathe-Lernspiel für Kinder (Klasse 1–6), das direkt im Browser läuft.

**▶️ [Jetzt spielen](https://johannesrabauer.github.io/l-die-zaubertafel/)**

## Features

- 🖍️ Grüne Schultafel mit realistischem Kreide-Effekt
- 📚 Aufgaben angepasst an Klassenstufen 1–6
- 🎮 Vier Spielmodi: Übung, Punkte, Zeit, Freies Schreiben
- ⭐ Motivierendes Belohnungssystem mit Animationen
- 📱 Funktioniert auf Computer, Tablet und Smartphone
- 🔒 Keine Installation, kein Tracking, keine externen Abhängigkeiten

## Technologie

Vanilla HTML5, CSS3 und JavaScript – kein Framework, kein Build-Schritt.

## Lokal starten

Beliebigen statischen Server verwenden, z.B.:

```bash
npx serve .
```
